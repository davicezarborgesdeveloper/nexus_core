Nexus Core Design System
Tokens, paleta de cores, tipografia, espaçamentos e catálogo de componentes do portfólio Nexus Core. Estética editorial: azul-marinho institucional como cor estrutural, dourado como acento pontual.
Cores
Paleta
Azul-marinho profundo como cor primária e estrutural. O dourado (--accent-color) é usado apenas como acento pontual, nunca como cor predominante.
